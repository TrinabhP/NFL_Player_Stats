# NFL_Player_Stats

Contributers: tponnapa@calpoly.edu, jcroney@calpoly.edu, pmoschit@calpoly.edu, caso@calpoly.edu

This API is a public record of NFL player combine statistics that tracks their performance data across events like the 40 yd dash, bench press, broad jump, etc. Users can write combine stat entries for individual players. The data can be analyzed to compare prospective players who are entered with similar players already in the NFL. This can be used to predict success based on if they display similar attributes to a star player.
